📱 Little Drop – Login UI (Jetpack Compose)

Little Drop is a mobile application developed in Kotlin using Jetpack Compose for creating modern Android interfaces.

The goal of this project is to implement an authentication interface with Home, Login, and Registration screens, following a modern design.

This project was developed in Android Studio using Jetpack Compose to build the declarative interface.

✨ Features

Modern interface built with Jetpack Compose

Home screen

Login screen

Register screen

Custom input fields

Buttons with rounded corners

Visual social media integration

Use of Material Design icons

Responsive design using Column, Row, and Box layouts

📷 App Screens

The app includes three main screens:

🏠 Home

Welcome screen where the user can:

Log in

Register

View social media login options

🔐 Login

Screen where the user can enter:

Email

Password

Also includes:

Password recovery

Social media login

📝 Register

Screen for creating a new account where the user enters:

Username

Email address

Password

Confirmation Password

It also includes a back button.

Language: Kotlin
